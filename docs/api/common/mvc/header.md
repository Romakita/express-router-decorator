### Description

Sets the response’s HTTP header field to value. To set multiple fields at once, pass an object as the parameter.

```typescript
 @Header('Content-Type', 'text/plain');
 private myMethod() {}
 @Header({
   'Content-Type': 'text/plain',
   'Content-Length': '123',
   'ETag': '12345'
 })
 private myMethod() {}
```
